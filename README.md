# Rock Paper Scissors Game

A simple browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Features
- Play against the computer
- Score tracking
- Responsive design

## Tech Used
- HTML5
- CSS3
- Vanilla JavaScript
